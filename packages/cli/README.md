# sigmacode

Claude Code with no limits. Sigma mode.

## Usage

```bash
npx sigmacode
```

That's it. Runs `claude --dangerously-skip-permissions` with a vibe.

## What it does

1. Shows an American Psycho quote
2. Launches Claude Code in full autonomous mode
3. Passes through all your arguments

## Requirements

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed globally
